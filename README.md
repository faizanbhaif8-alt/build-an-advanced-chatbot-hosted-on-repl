# Advanced Chatbot

This is an advanced chatbot built with a modern VIP-style UI. It supports text chat, file uploads, and real-time voice processing.

## Features
- Chat with the bot using text input
- Upload files for processing
- Record and process live audio
- Modern VIP-style UI with navigation bar and sidebar

## Setup
1. Clone the repository
2. Install dependencies
3. Set up Replit Secrets with API keys
4. Run the application
